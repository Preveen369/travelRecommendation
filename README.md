# Travel Recommendation
Travel Recommendation is a static web application that assists users in discovering personalized travel destinations based on their preferences, such as interests, budget, and travel season. It leverages mock API data from a local JSON file to generate dynamic suggestions, complete with destination details, images, and itineraries, all powered by vanilla web technologies for instant, offline access.
